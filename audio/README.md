## How to Add Your Audio:

1. Record your voice messages
2. Convert to MP3 format (128kbps recommended)
3. Name them according to the list above
4. Place them in this folder
5. Test by running: npm run dev

For detailed recording instructions, see AUDIO_GUIDE.md in the root folder.

Note: The app will work without audio files (text-only mode), but audio adds a magical personal touch!
